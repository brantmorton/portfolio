ABOUT
I began learning Python in college as a tool to solve higher order chemical engineering problems. I got hooked to the challenges that learning how to code brought. That snowballed, and developed into a passion for building things with code.

My language experience is in Python and Javascript, and my framework knowledge lies in Django/Django Rest and React. However, I believe the principles of programming span deeper than the tools we use, and I welcome the challenge of learning a new stack if necessary.

PROJECTS

Civil Construction Company

I built a web app for a local construction company. Per the client's request, the project includes an application form that stores applicant information to a database and emails HR the applicant's submitted data.

Backend: Django, PostgresQL, Apache

Frontend: HTML5, CSS

Excused - Excuse Generator

This project generates random excuses for the user. It is built using React Hooks on the frontend, with a custom API built on Django Rest Framework for the backend. The app has the functionality to display categorized excuses that are sorted client side.

Backend: Django Rest Framework, Heroku

Frontend: React

Portfolio Site

This is an on-going project to display my full-stack work. I built the full UI with custom HTML and CSS. The header is using Particles.js, a Javascript library. The contact section is powered by Django.

Backend: Django, Apache

Frontend: HTML5, CSS, Javascript
